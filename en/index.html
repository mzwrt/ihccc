About Us

Nonprofit Organization · Committed to Internet Security & Information Compliance

IHCCC (China Internet Security & Compliance Center) is a nonprofit organization dedicated to advancing standardization, regulation, and best practices in internet security and information compliance. Our core mission is to provide professional security compliance solutions to society, enterprises, and institutions, helping them comply with applicable laws and regulations in digital environments while ensuring data security and network protection.

Internet Security

Enhancing the security of network infrastructure, defending against various cyberattacks, and ensuring the security and integrity of information transmission.

Information Compliance

Helping organizations understand and comply with domestic and international regulations on data protection, privacy rights, and compliance requirements such as GDPR and China's Cybersecurity Law.

Standardization & Certification

Promoting and assisting in the implementation of industry standards and certification frameworks, ensuring organizations meet international requirements for information security and compliance.

Core Standards

China Standards & International Security Standards

We integrate China's cybersecurity and information security laws and standards (MLPS 2.0, Cybersecurity Law, Data Security Law, PIPL, Cryptography Law, etc.) with international security standards (CIS, PCI-DSS, ISO 27001, NIST CSF, SOC 2), providing authoritative guidance and compliance resources.

IHCCC-CS

IHCCC-CS Cybersecurity Standard

IHCCC Cybersecurity Standards

IHCCC-CS is a cybersecurity standards framework independently developed by IHCCC. Drawing on multiple international standards and tailored to the realities of Chinese internet enterprises, it provides a practical and implementable security governance roadmap.

  • Independently developed by IHCCC, not a simple framework mashup
  • References CIS, NIST, ISO 27001, MLPS 2.0, and other standards
  • Tailored for Chinese internet enterprises, highly practical

CIS

CIS Security Benchmarks

Center for Internet Security Benchmarks

CIS Benchmarks are the globally recognized best-practice standards for secure configuration, covering operating systems, cloud platforms, network devices, databases, and more.

  • OS security benchmarks (Windows, Linux, macOS)
  • Cloud platform benchmarks (AWS, Azure, GCP, Alibaba Cloud)
  • Container & orchestration security (Docker, Kubernetes)

PCI-DSS

PCI-DSS Payment Security

Payment Card Industry Data Security Standard

PCI-DSS is the global standard for protecting cardholder data, applicable to all organizations that store, process, or transmit payment card data.

  • Full text and guidance for PCI-DSS v4.0
  • In-depth coverage of 12 core security requirements
  • Compliance implementation roadmap & best practices

ISO 27001

ISO 27001 Information Security

Information Security Management System

ISO 27001 is the internationally recognized standard for information security management systems, helping organizations systematically manage information security risks.

  • ISMS (Information Security Management System) framework
  • Annex A controls guidance & interpretation
  • Certification process guide & best practices

NIST CSF

NIST Cybersecurity Framework

NIST Cybersecurity Framework

The NIST Cybersecurity Framework provides a risk-based approach to managing cybersecurity, widely adopted by organizations of all sizes for building security programs.

  • Five core functions: Identify, Protect, Detect, Respond, Recover
  • NIST SP 800 series documentation & guidance
  • Risk assessment methodology & implementation guidance
Learn More
SOC 2

SOC 2 Audit Standard

System and Organization Controls 2

SOC 2 is the audit standards framework for assessing service organizations on security, availability, processing integrity, confidentiality, and privacy.

  • Trust Services Criteria (TSC) guidance & interpretation
  • Type I and Type II audit guidance
  • Compliance readiness & continuous monitoring plans
Learn More
MLPS 2.0

Multi-Level Protection Scheme

GB/T 22239-2019 Information Security Technology – Baseline for Classified Protection of Cybersecurity

MLPS 2.0 is the core standard of China's Multi-Level Protection Scheme, applicable to all network operators and a foundational cybersecurity compliance requirement for enterprises.

  • Five-level classification & grading guidance
  • General & extended security requirements explained
  • MLPS evaluation process & remediation guidance

Cybersecurity & Data Protection Laws

Cybersecurity Law · Data Security Law · Personal Information Protection Law

China's three foundational cybersecurity and data protection laws form the legal compliance framework for enterprises and are mandatory for all network operators and data processors.

  • Key provisions of the Cybersecurity Law explained
  • Data classification, grading & security review under the Data Security Law

Crypto

Commercial Cryptography Security

Cryptography Law & GM/T 0054 – Commercial Cryptography Application Security Evaluation

China's Cryptography Law and Commercial Cryptography Application Security Evaluation (Crypto Evaluation) are the core regulations and standards for cryptography management, applicable to information systems using commercial cryptography.

  • Core requirements & compliance obligations under the Cryptography Law
  • Commercial cryptography application security evaluation process
  • Cryptographic technology application plans & best practices

CIIP

Critical Information Infrastructure Protection

Regulations on Critical Information Infrastructure Security Protection & Related Standards

Critical Information Infrastructure Protection is a top priority in China's cybersecurity regime, covering CIIP designation, security protection measures, security assessment, and more.

  • CIIP designation criteria & protection obligations
  • Security protection measures & emergency response requirements
  • Security assessment guide & compliance pathway

GB/T

National Standards GB/T Series

National Information Security Technology Standards System

China's national information security technology standards (GB/T) series covers risk assessment, security auditing, data security, privacy protection, and more — an important reference for compliance programs.

  • GB/T 20984 – Information Security Risk Assessment Specification
  • GB/T 35273 – Personal Information Security Specification
  • GB/T 37988 – Data Security Capability Maturity Model
